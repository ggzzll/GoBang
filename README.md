# 五子棋API
## 项目介绍  
该代码使用PHP代码编写，使用了GD库与宋体字体  
项目地址：https://github.com/ggzzll/GoBang  
## 使用方法  
/?id=用户id  
加入五子棋棋局(满两人自动开始游戏)  
/?out=用户id  
退出五子棋棋局(仅限加入五子棋的玩家)  
/?id=用户id&x=坐标x轴&y=坐标y轴  
五子棋下棋(只能下到空白处，无法下到棋盘外)
